# 智慧製造

Smart Manufacturing，工業4.0 的核心實踐方法。

## 定義

智慧製造是一種先進的製造模式，透過整合物聯網、人工智慧、大數據分析等技術，實現生產過程的自動化、智慧化與最佳化。

核心特徵：**整合思維** > 單一技術

## 與傳統製造的差異

| 面向 | 傳統製造 | 智慧製造 |
|------|---------|---------|
| 數據收集 | 人工記錄 | 感測器自動採集 |
| 決策方式 | 經驗判斷 | 數據驅動 |
| 問題發現 | 事後檢查 | 即時監測 |
| 維護策略 | 定期保養 | 預測性維護 |
| 生產彈性 | 固定產線 | 彈性調整 |

## 核心要素

### 1. 連接（Connectivity）

設備互聯互通：

```
感測器 ←→ PLC ←→ MES ←→ ERP ←→ 雲端
   ↓
ROS2/MQTT（你的專案）
```

### 2. 數據（Data）

| 數據類型 | 說明 | 範例 |
|----------|------|------|
| 設備數據 | 機台運行狀態 | 溫度、振動、轉速 |
| 製程數據 | 生產參數 | 良率、週期時間 |
| 品質數據 | 產品檢測結果 | 尺寸、外觀 |
| 環境數據 | 廠房條件 | 濕度、空氣品質 |

### 3. 分析（Analytics）

```
描述性分析：發生了什麼？ → Dashboard
    ↓
診斷性分析：為什麼發生？ → 根因分析
    ↓
預測性分析：會發生什麼？ → 預測模型
    ↓
處方性分析：該怎麼做？   → AI 建議
```

### 4. 自動化（Automation）

- **物理自動化**：機器人、AGV
- **流程自動化**：自動排程、自動補貨
- **決策自動化**：AI 驅動的即時調整

## 關鍵績效指標（KPI）

### 設備效率

| 指標 | 全名 | 計算方式 | 目標 |
|------|------|---------|------|
| OEE | 設備綜合效率 | 可用率 × 效能 × 良率 | >85% |
| MTBF | 平均故障間隔 | 總運行時間 / 故障次數 | 越高越好 |
| MTTR | 平均修復時間 | 總維修時間 / 維修次數 | 越低越好 |

### 維護指標

| 指標 | 說明 |
|------|------|
| 計畫性維護比例 | 預防性 / 總維護次數 |
| 備件週轉率 | 庫存效率 |
| 維護成本比 | 維護費用 / 設備價值 |
