# 工廠感測器與監測指標

工廠數位孿生中常用的感測器類型與監測指標說明。

## 常見感測器類型

### 溫度感測器（Temperature）

**用途**：監測設備運行溫度，預防過熱故障

| 異常狀況 | 可能原因 |
|----------|---------|
| 溫度過高 | 潤滑不足、摩擦過大、散熱不良 |
| 溫度驟升 | 軸承損壞、馬達過載 |
| 溫度波動大 | 負載不穩、冷卻系統異常 |

**你的專案**：
```python
"temperature": round(random.uniform(60.0, 95.0), 1)
```

**業界標準範圍**（旋轉設備）：
| 等級 | 溫度 | 狀態 |
|------|------|------|
| 正常 | < 70°C | 綠燈 |
| 警告 | 70-85°C | 黃燈 |
| 危險 | > 85°C | 紅燈 |

---

### 振動感測器（Vibration）

**用途**：機械健康狀態的核心指標

> 「如果溫度是機器的『發燒』，振動就是機器的『心跳』」

| 振動模式 | 可能原因 |
|----------|---------|
| 低頻振動 | 不平衡、軸心偏移 |
| 高頻振動 | 軸承磨損、齒輪問題 |
| 衝擊振動 | 鬆動、裂紋 |

**你的專案**：
```python
"vibration": round(random.uniform(0.1, 5.0), 2)
```

**ISO 10816 振動標準**（mm/s RMS）：
| 等級 | 振動值 | 狀態 |
|------|--------|------|
| A（優良） | < 1.8 | 新機或大修後 |
| B（可接受） | 1.8 - 4.5 | 正常運行 |
| C（勉強） | 4.5 - 11.2 | 需安排維護 |
| D（不可接受） | > 11.2 | 立即停機 |

---

### 狀態指示（Status）

**用途**：設備運行狀態的綜合判斷

**你的專案**：
```python
"status": random.choice(["running", "warning", "error"])
```

**建議擴充狀態**：
| 狀態 | 說明 | 顏色 |
|------|------|------|
| `idle` | 待機中 | 灰 |
| `running` | 正常運行 | 綠 |
| `warning` | 需注意 | 黃 |
| `error` | 故障 | 紅 |
| `maintenance` | 維護中 | 藍 |

## 其他常見感測器

### 電流感測器（Current）

| 異常 | 意義 |
|------|------|
| 電流過高 | 過載、卡住 |
| 電流不穩 | 負載變化、機械問題 |
| 啟動電流異常 | 啟動困難 |

```python
# 建議加入
"current": round(random.uniform(2.0, 15.0), 1)  # 安培
```

### 壓力感測器（Pressure）

適用於：液壓系統、氣壓系統、管路

```python
"pressure": round(random.uniform(1.0, 10.0), 1)  # bar
```

### 轉速感測器（RPM）

```python
"rpm": random.randint(1000, 3000)
```

### 運行時數（Runtime）

用於計算 MTBF、排程維護

```python
"runtime_hours": 1234.5
```

## 預測性維護指標

### 健康度分數（Health Score）

綜合多個感測器計算：

```python
def calculate_health(temp, vibration):
    temp_score = 100 - ((temp - 60) / 35 * 100)  # 60-95 → 100-0
    vib_score = 100 - (vibration / 5.0 * 100)     # 0-5 → 100-0
    return round((temp_score + vib_score) / 2, 1)
```

PUBLISH_INTERVAL = 1.0  # 每秒發布一次

## 告警閾值設計

### 多層級告警

```python
THRESHOLDS = {
    "temperature": {
        "warning": 75.0,
        "critical": 85.0,
    },
    "vibration": {
        "warning": 3.0,
        "critical": 4.5,
    }
}

def check_alert(data):
    alerts = []
    if data["temperature"] > THRESHOLDS["temperature"]["critical"]:
        alerts.append({"level": "critical", "msg": "溫度過高！"})
    elif data["temperature"] > THRESHOLDS["temperature"]["warning"]:
        alerts.append({"level": "warning", "msg": "溫度偏高"})
    return alerts
```

### 狀態權重

```python
STATUS_WEIGHTS = {
    "running": 0.6,   # 60% 機率正常
    "warning": 0.3,   # 30% 機率警告
    "error": 0.1      # 10% 機率錯誤
}

# 使用 random.choices 實現
status = random.choices(
    list(STATUS_WEIGHTS.keys()),
    weights=list(STATUS_WEIGHTS.values())
)[0]
```
